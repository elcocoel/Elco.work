# elco.work (marketing)

Next.js site for [elco.work](https://elco.work). Local dev: `npm run dev:stable` (see `package.json`).

## GitHub (new remote)

1. Create an **empty** repo on GitHub (no README/license) under your account.
2. From this directory:

```bash
git remote add origin https://github.com/YOUR_USER/YOUR_NEW_REPO.git
git push -u origin main
```

Or with GitHub CLI: `gh repo create YOUR_NEW_REPO --private --source=. --remote=origin --push`
